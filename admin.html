使用说明

  1. 在上方配置区填写您的手机号码及品牌信息
  2. 打开 index.html，修改顶部 CONFIG 对象中的 phone 等字段
  3. 将 index.html 上传到 Web 服务器，获取公网 URL
  4. 将公网 URL 粘贴到上方「落地页 URL」输入框，点击生成二维码
  5. 点击打印 / 导出，将二维码打印出来张贴或分发
提示：如无公网服务器，可将 index.html 放入任意静态托管服务（如 GitHub Pages、Vercel、Netlify、阿里云 OSS 等），获取 HTTPS 地址后再生成二维码。
